# finding_dif_words

This repository contains code used in the paper:

***"Finding Words Associated with DIF: Predicting Differential Item Functioning using LLMs and Explainable AI"***

***by Hotaka Maeda & Yikai (EK) Lu.***

In our paper, we ran scripts numbered 0_ to 5_ in order. Scripts 2_ and 4_ are the most important for replicating our 3-category modeling approach. Data are not included as they are confidential.
